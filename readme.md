## ⑤工夫した点・こだわった点
- 入力時に自分の名前を入れることで、無責任な発言がないように促します（管理者画面では全情報を閲覧できるので、不規則発言をする人がいた場合はわかるようになっています）。
- フィードバック内容は分析シートで全員分を確認することができます。自分の作品に向けられた評価だけでなく、ほかの人の作品に対して周囲がどのような評価をしたかを閲覧でき、多様な意見に学べるような構成にしました。

## ⑥難しかった点・次回トライしたいこと（又は機能）
- 今回は課題提出用に発表者をプリセットしていますが、自由度が低いので、「参加者がログインする、ログイン情報が参加者リストになる、参加者リストが発表者のプルダウンに反映される」という機能を追加できると良いのではと思っています。
- 毎度感じることなのですが、最初に課題の要件を満たすシンプルなコードを書き、そこに機能を少しずつ追加していくと、徐々にどのコードが何に効いているのかが分からなくなっていきます。大きな課題です。
